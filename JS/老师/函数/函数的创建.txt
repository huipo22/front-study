一、函数
  概念：将 某一功能的代码  集合 起来，方便我们以后能 重复调用  的 代码块 就是函数。

二、函数的优点：
  C 使程序更加简洁
  C 逻辑性更调理
  C 调用更方便
  C 维护更加容易
   - 提高工作效率（更加省事）

三、函数的定义方式
  1.关键字 function   var num=23
   结构：function 函数名 (){
             执行的代码;
         }
  2.匿名函数(直接量，字面量的形式)
   var aa=function (){
            函数要执行的代码;
        }
   字面量：赋值符号右侧的东西，可以是任何的数据类型。（有本身的意义）例：12  函数
   3.通过对象来定义(对象：一切皆对象)
   使用new关键字 
   结构:  var aa=new Function("[参数1]"，"[参数2]",...,"函数要执行的代码;(函数体)");

四、函数的调用方式
   1.函数名（）;
     作用场合：1.具名函数（有名字的函数）
           2.匿名函数赋值给一个变量，调用这个函数也可以用函数名();

   2.自调用:专用与匿名函数的调用
       1.(function(){
    	    alert(1);
          })()
       第一个括号:提高函数的优先级
       第二个括号：运行
       2. (function(){
    	    alert(1);
          }())
       第一个括号：运行
       第二个括号：强制输出函数的运行结果
   3.通过触发事件来调用(不常用)
       <div class="box" onclick="aa();"></div>

  
五、function与字面量形式定义的函数的区别

  1.如果函数名相同，后面的函数会覆盖前面的函数

  2.使用function关键字来定义的函数，会提前解析到内存中,我们调用时，可以在它前面调用。
    使用字面量的形式来定义的函数，只有执行到它自已时，才会将函数赋值给这个变量，我们调用时，只能在它的后面调用。  

   3.javascript解析时是按一个一个的script标签对来解析的,只有上面的script标签对解析完成以后，才会解析下一个script标签对。
     先定义  后调用。



六、函数的参数
   存在的意义：可以动态的改变函数内部的值，从而使一个函数实现不同的效果。

   形参：函数在定义时括号里传入的值，用来接受实参的。
   实参：函数在调用时括号里传入的值，用来改变形参的。   

七、参数使用时需要注意什么：
   
   1.参数可以是任何的数据类型。
   //一个可以识别数据类型的函数
   
   2.参数的个数(最多25个)
      A.形参与实参个数相同，一一对应。
   
      B.实参个数大于形参,怎么获取多出来的实参。(arguments)
      
      C.实参个数小于形参，形参会自动赋值为undefined。
     
八、arguments对象
   当定义一个函数时，会隐式的创建一个arguments对象，它用来存储函数的参数的信息.
   属性：
   1.length  长度，函数参数的长度
   2.callee  调用函数本身
   arguments的下标来获取具体的实参的值(从0开始的)
   例：arguments[3] 第四个
   
九、return
   1.终止并跳出这个函数的
      a.如果return后面有代码，是不会执行的。
      sb.一个函数可以有多个return语句，但是只会一个return执行。（一般与判断语句一起使用）。   
   2.给函数返回一个值
      a.如果函数没有返回值时，会自动赋值为undefined
      b.一个函数只会有一个返回值。
      c.返回值可以是任何数据类型。
